# Tic Tac Toe Game

A simple Tic Tac Toe game built with HTML, CSS, and JavaScript.

## How to Play

- Click on the cells to place your X or O.
- The first player to get three in a row wins.
- If all cells are filled without a winner, it's a draw.
- Click "Reset Game" to start over.

## Running the Game

Open `index.html` in your web browser to play the game.

## Adding to iOS as an App

To add this game as a web app on your iPhone:

1. Open the game in Safari on your iPhone.
2. Tap the share button (square with arrow).
3. Select "Add to Home Screen".
4. Name it and tap "Add".

This will create an icon on your home screen that opens the game like a native app.
